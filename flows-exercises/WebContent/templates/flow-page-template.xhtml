Navigate to Main Flow Pages

Only the links that apply to current flow should work. The links that apply to the other flow should result in error messages. However, when using CDI with Tomcat via the Weld servlet, the system does not properly enforce that you cannot use an outcome referring to a page outside the flow other than yourflow-return. Glassfish and other full Java EE 7 servers enforce this correctly.
